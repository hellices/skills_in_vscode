# Java Example Project

Example Java/Spring Boot project with GitHub Copilot custom instructions and Anthropic skills.

## Setup

```bash
./mvnw clean install
```

```
├── src/main/java/com/example/

│   └── service/

├── .vscode/
│   └── skills.json
└── pom.xml
```
